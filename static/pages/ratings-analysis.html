Ratings Analytics

Ratings Analytics provides an organic user experience for data geeks to gain insights about their favorite movies.

This app uses Apache Spark to process a file containing 20 million movie ratings (to learn more about the data, click here) and presents the findings of this analysis in a way that is intuitive for the user. On the home screen (Figure 1) the user can either search for their desired movie or click on a link to the movies tab (Figure 2) and browse all movies in alphabetical order.

The page for each movie visualizes a summary of the movie ratings for that movie. The word cloud (left) shows all of the tags that users associated with the movie while the pie chart (right) uses tooltips to show the number of each rating submitted by the users (Figure 3). The final chart (Figure 4) shows the breakdown of ratings over time.

Technical Stack: Apache Spark, Spark SQL, Redis, Python, Flask, JavaScript, D3.js, C3.js, jQuery, AJAX, HTML/CSS, Bootstrap
